Spring Free Sample Web App

This is a very simple web app that can serve as a starting place for developing a OSGi based web application without any Spring Framework usage. This sample uses a regular servlet container and is deployed as a 'Web Application Bundle', an alternative would be to use the OSGi HTTPService. This can run on any Virgo distribution and may be more suitable especially for more lightweight applications. From this sample page you can access the web-based admin console.
